# Fruit Recognizer

This is a simple android flutter project. A camera application that recognizes fruits.


```
dependencies:
  flutter:
    sdk: flutter

  camera: ^0.10.6
  cupertino_icons: ^1.0.6
  image_picker: ^1.1.1
  tflite: ^1.1.2
```

Works without errors with `Flutter 3.19.3`. Be aware `tflite` is not compitable with newer versions.
